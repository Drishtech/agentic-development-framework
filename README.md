# agentic-development-framework
Agentic Development Framework (ADF) estandariza el desarrollo de software por agentes LLM (8B-32B). Elimina la pérdida de contexto y pereza técnica forzando TDD (Red/Green) y validación mecánica de invariantes arquitectónicas. Garantiza código production-ready limitando el contexto y bloqueando el progreso sin evidencia ejecutable.
